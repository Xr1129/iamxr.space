## 部署备忘

每次编辑完内容后，执行以下命令更新网站：

```bash
cd /var/www/iamxr.space && git pull && npm run build && systemctl restart iamxr.space
```

一些注意事项：
- 确保 `git pull` 前本地修改已提交并推送
- `npm run build` 会生成 `.next/` 构建产物
- `systemctl restart iamxr.space` 重启服务以应用新版本
